## How to Use

1. Open the notebook:
   ```bash
   jupyter notebook simple_notebook.ipynb
   ```

   Or use JupyterLab (more modern interface):
   ```bash
   jupyter lab
   ```

2. This will open in your browser automatically

3. Run cells by clicking them and pressing `Shift+Enter`

## What's Inside

The notebook includes:
- Simple list adder function
- List operations
- Basic calculations
- Data visualizations (bar charts, line plots)
- Pandas DataFrames
- Interactive examples
